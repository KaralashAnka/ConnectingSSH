Задание 2: Настройка SSH-ключей и псевдонима

Скриншот 2 (Снимок2.PNG):

Создание SSH-ключа командой ssh-keygen -t rsa -b 4096 и копирование публичного ключа на сервер. После настройки выполнено успешное подключение к серверу без ввода пароля.

Скриншот 3 (Снимок3.PNG):

Содержимое файла конфигурации SSH (~/.ssh/config). Настроен псевдоним "netology" для быстрого подключения к серверу с IP 89.111.170.114 от пользователя root с использованием SSH-ключа.

Скриншот 4 (Снимок4.PNG):

Успешное подключение к серверу по псевдониму командой ssh netology. Подключение выполнено без запроса пароля благодаря настроенным SSH-ключам и файлу конфигурации.
